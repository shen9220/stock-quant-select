# 股票量化模型一键运行工具

## 快速开始

### Windows用户

1. 双击 `启动模型.bat` 文件
2. 选择要运行的模型（1-6）
3. 选择日期（1-3）
4. 查看运行结果

### Linux/Mac用户

```bash
# 方法1：双击或双击 `启动模型.sh` 文件

# 方法2：命令行运行
./启动模型.sh

# 方法3：直接运行Python脚本
python3 run_models.py
```

### 所有平台通用

```bash
python run_models.py
```

## 使用说明

### 主菜单

运行后会显示以下菜单：

```
======================================================================
                    股票量化模型一键运行工具
======================================================================

请选择要运行的模型：

  [1] 强度首板战法
      首次涨停股票分析，早盘集合竞价买入，次日收盘前卖出

  [2] 涨停三阴法
      首次涨停后阴跌战法，涨停后3天阴跌不破底

  [3] 隔夜战法（深度）
      深度选股模型，买阴不买阳策略

  [4] 隔夜战法（股票）
      隔夜选股模型

  [5] 三阴战法
      三阴战法选股模型

  [6] 运行所有模型
      依次运行所有量化模型

  [0] 退出程序
======================================================================
```

### 选择模型

输入数字 `1-6` 选择要运行的模型：
- `1`：运行强度首板战法
- `2`：运行涨停三阴法
- `3`：运行隔夜战法（深度）
- `4`：运行隔夜战法（股票）
- `5`：运行三阴战法
- `6`：依次运行所有模型
- `0`：退出程序

### 选择日期

选择模型后会提示选择日期：

```
请选择日期：
  [1] 今天（默认）
  [2] 昨天
  [3] 自定义日期（格式：YYYY-MM-DD）

请输入选项（1-3，默认1）:
```

- 输入 `1` 或直接回车：使用今天的日期
- 输入 `2`：使用昨天的日期
- 输入 `3`：手动输入日期，格式为 `YYYY-MM-DD`（例如：2024-04-24）

### 查看结果

程序会自动运行选中的模型，并在控制台输出结果。结果包括：

1. 模型分析信息
2. 所有股票的评分
3. 前2名/前5名的详细分析
4. 交易建议（买入价、止盈价、止损价）
5. JSON格式结果文件

### 继续运行

运行完一个模型后，会询问是否继续：

```
是否继续运行其他模型？（y/n，默认y）:
```

- 输入 `y` 或直接回车：返回主菜单继续运行其他模型
- 输入 `n`：退出程序

## 模型详细说明

### 1. 强度首板战法

**文件**：`首板战法/up.py`

**策略**：
- 连续涨停1天（非连板）
- 早盘集合竞价买入
- 次日收盘前全部卖出
- 仓位：50%

**输出**：前2名股票的详细分析

**结果文件**：`首板战法/strength_limit_up_result.json`

### 2. 涨停三阴法

**文件**：`涨停三阴法/pop3.py`

**策略**：
- 首次涨停后连续3天阴跌不破涨停底部
- 第3或4天收十字星
- 集合竞价买入半仓第一名
- 次日清仓

**输出**：前5名股票的详细分析

**结果文件**：`涨停三阴法/yin_die_result.json`

### 3. 隔夜战法（深度）

**文件**：`隔夜战法/deep.py`

**策略**：
- 买阴不买阳
- 历史放量大阳线
- 缩量阴线
- 回调10日线

**输出**：符合条件的股票分析

**结果文件**：根据配置

### 4. 隔夜战法（股票）

**文件**：`隔夜战法/stock.py`

**策略**：隔夜选股策略

**输出**：选股结果

### 5. 三阴战法

**文件**：`三阴战法/pop3.py`

**策略**：三阴战法选股

**输出**：选股结果

## 命令行参数

如果您想直接使用命令行运行单个模型，可以使用以下方式：

### 强度首板战法

```bash
python 首板战法/up.py --date 2024-04-24 --top-n 2 --output result.json
```

参数说明：
- `--date`：分析日期（YYYY-MM-DD）
- `--top-n`：返回前N只股票（默认2）
- `--output`：输出文件路径

### 涨停三阴法

```bash
python 涨停三阴法/pop3.py --date 2024-04-24 --top-n 5 --output result.json
```

参数说明：
- `--date`：分析日期（YYYY-MM-DD）
- `--top-n`：返回前N只股票（默认5）
- `--output`：输出文件路径

### 其他模型

```bash
python 隔夜战法/deep.py --date 2024-04-24
python 隔夜战法/stock.py --date 2024-04-24
python 三阴战法/pop3.py --date 2024-04-24
```

## 常见问题

### Q1: 双击批处理文件没有反应？

**解决方案**：
1. 确保已安装Python并添加到环境变量
2. 右键点击批处理文件，选择"编辑"
3. 将第一行的 `python` 改为 `python3` 或完整的Python路径
4. 保存后重新运行

### Q2: Linux/Mac无法运行.sh文件？

**解决方案**：
```bash
# 添加执行权限
chmod +x 启动模型.sh

# 然后运行
./启动模型.sh
```

### Q3: 运行时提示找不到Python？

**解决方案**：
1. 检查是否安装Python：`python --version` 或 `python3 --version`
2. 如果未安装，请从 https://www.python.org 下载安装
3. 确保Python已添加到系统环境变量

### Q4: 运行模型时出错？

**解决方案**：
1. 检查模型文件是否存在
2. 查看错误信息，确认是否缺少依赖库
3. 安装必要的依赖：`pip install akshare pandas numpy`

### Q5: 如何修改默认日期？

**解决方案**：
- 在菜单中选择自定义日期选项（选项3）
- 输入您想要的日期，格式为 `YYYY-MM-DD`
- 例如：`2024-04-24`

### Q6: 结果文件保存在哪里？

**解决方案**：
- 结果文件保存在各模型目录下
- 强度首板战法：`首板战法/strength_limit_up_result.json`
- 涨停三阴法：`涨停三阴法/yin_die_result.json`
- 可以使用任何文本编辑器或JSON查看器打开

## 注意事项

1. **网络连接**：模型需要访问互联网获取股票数据，请确保网络连接正常
2. **运行时间**：首次运行可能需要下载依赖包，请耐心等待
3. **数据时效**：建议在交易日收盘后运行，数据更准确
4. **风险提示**：模型仅供参考，不构成投资建议，请谨慎决策
5. **版本兼容**：需要Python 3.6或更高版本
