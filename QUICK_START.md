# 快速开始指南

## 运行方式

### 方式1：使用启动脚本（推荐）

#### Linux/Mac用户
```bash
# 进入项目目录
cd /workspace/projects/stock-quant-select

# 运行启动脚本
./启动模型.sh
```

#### Windows用户
```bash
# 双击运行
启动模型.bat
```

### 方式2：直接运行Python脚本

```bash
# 进入项目目录
cd /workspace/projects/stock-quant-select

# 运行主程序
python3 run_models.py
```

## 使用步骤

1. **运行程序**：选择上述任一方式运行程序
2. **选择模型**：输入数字 1-6 选择要运行的模型
   - 1: 强度首板战法
   - 2: 涨停三阴法
   - 3: 隔夜战法（深度）
   - 4: 隔夜战法（股票）
   - 5: 三阴战法
   - 6: 运行所有模型
3. **选择日期**：
   - 1: 今天
   - 2: 昨天
   - 3: 自定义日期（格式：YYYY-MM-DD）
4. **查看结果**：程序自动运行并显示分析结果
5. **继续或退出**：输入 y 继续，输入 n 退出

## 常见问题

### Q1: 找不到文件？
**答**：确保您在正确的目录下运行
```bash
cd /workspace/projects/stock-quant-select
```

### Q2: 提示权限错误？
**答**：给启动脚本添加执行权限
```bash
chmod +x 启动模型.sh
```

### Q3: 提示找不到Python？
**答**：确保已安装Python3
```bash
python3 --version
```
