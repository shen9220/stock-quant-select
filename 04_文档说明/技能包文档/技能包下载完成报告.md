## 💡 在量化项目中的应用

### 应用场景

1. **生成选股报告音频**
   - 使用 `coze generate audio` 生成选股结果的语音播报
   - 参考：`coze-generate-audio.md`

2. **生成K线图和分析图表**
   - 使用 `coze generate image` 生成K线图、趋势图
   - 参考：`coze-generate-image.md`

3. **上传分析结果到云端**
   - 使用 `coze file upload` 上传分析结果文件
   - 获取在线可访问链接
   - 参考：`coze-file-upload.md`

4. **管理量化项目**
   - 使用 `coze code project` 创建和管理量化项目
   - 使用 `coze code deploy` 部署应用
   - 参考：`coze-code-project.md`, `coze-code-deploy.md`

### 示例代码

```bash
# 生成选股报告图片
coze generate image --prompt "股票K线图分析" --output stock_chart.png

# 上传文件到云端
coze file upload --file stock_chart.png

# 生成语音播报
coze generate audio --text "今日推荐股票：平安银行，买入价12.50元"
```
