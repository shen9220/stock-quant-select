# Stock Assistant - 最简单的使用指南

## 🎯 这是什么？

一个智能选股助手，可以帮你：
- ✅ 选股票（按行业、市值、估值筛选）
- ✅ 分析个股（给评分、提建议）
- ✅ 找热点（推荐热门板块）

---

## ⚡ 怎么用？（超简单！）

### 第一步：加载技能
直接说：
```
加载 stock-assistant 技能
```

### 第二步：开始提问
```
帮我选几只银行股
```

```
000001平安银行怎么样？给个评分
```

```
最近什么板块比较热门？
```

```
帮我找几只高股息的股票
```

---

## 📝 常用问题模板

### 选股类
```
帮我选几只低估值的[行业]股
```
```
我想买[行业]，有什么推荐的吗？
```
```
帮我找几只[市值]盘股
```

### 分析类
```
[股票代码]怎么样？给个评分
```
```
这只股票能买吗？帮我分析一下
```

### 技术类
```
帮我找几只技术面金叉的股票
```
```
有超跌反弹的机会吗？
```

## 🎯 评分标准

- ⭐⭐⭐⭐⭐ (85-100分): 强烈推荐买
- ⭐⭐⭐⭐ (70-84分): 可以买
- ⭐⭐⭐ (55-69分): 再看看
- ⭐⭐ (40-54分): 谨慎点
- ⭐ (0-39分): 别买
