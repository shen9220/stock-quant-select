## 🚀 快速开始（3步搞定）

### 第一步：进入目录

```bash
cd /workspace/projects/stock-quant-select/01_
./启动模型.sh
python up.py

# 选择 0-6 任意选项
# 选择日期
# 查看结果
```

### 第二步：运行程序

**Linux/Mac用户：**
```bash
./启动模型.sh
```

**Windows用户：**
```bash
双击 "启动模型.bat" 文件
```

**所有平台通用：**
```bash
python3 run_models.py
```

### 第三步：选择模型

根据菜单提示，输入数字选择要运行的模型：
- 输入 `1`：运行强度首板战法
- 输入 `2`：运行涨停三阴法
- 输入 `3`：运行隔夜战法（深度）
- 输入 `4`：运行隔夜战法（股票）
- 输入 `5`：运行三阴战法
- 输入 `6`：运行所有模型
- 输入 `0`：退出程序

## 🔧 运行要求

- 确保网络连接稳定

## 💡 使用技巧

### 技巧1：创建桌面快捷方式
- Windows：双击 `启动模型.bat` → 右键 → 发送到桌面快捷方式
- Linux/Mac：右键 `启动模型.sh` → 创建链接 → 拖到桌面

### 技巧2：运行所有模型
选择选项 `6` 可以一次性运行所有5个模型

### 技巧3：自定义日期
选择日期选项 `3`，输入格式：`YYYY-MM-DD`（如：2024-04-24）

### 技巧4：查看详细结果
结果文件保存在各模型目录下，格式为JSON，可用文本编辑器查看

## ⚠️ 注意事项

1. **目录位置**：确保在 `/workspace/projects/stock-quant-select/` 目录下运行
2. **权限问题**：Linux/Mac用户可能需要执行 `chmod +x 启动模型.sh`
3. **网络连接**：需要稳定的网络连接获取股票数据
4. **数据时效**：建议在交易日收盘后运行

## 🆘 常见问题

### Q1: 提示找不到文件？
**A**: 确保在正确的目录下
```bash
cd /workspace/projects/stock-quant-select
ls -la run_models.py
```

### Q2: 提示权限错误？
**A**: 添加执行权限
```bash
chmod +x 启动模型.sh
```

### Q3: 提示找不到Python？
**A**: 安装Python3或使用完整路径
```bash
# 检查Python
python3 --version

# 或使用完整路径
/usr/bin/python3 run_models.py
```

### Q4: 模型运行失败？
**A**: 检查依赖库和网络连接
```bash
# 检查依赖
pip list | grep -E "akshare|pandas|numpy"

# 安装缺失的库
pip install akshare pandas numpy
```
