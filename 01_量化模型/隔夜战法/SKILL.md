# 量化选股模型
运行：python scripts/stock_quant_select.py --date 2026-04-24

## 任务目标
本 Skill 用于执行量化选股策略，通过多技术指标筛选，每日从全市场A股中选出最符合策略的单一股票。

核心策略：买入缩量回调至10日线的股票，要求历史曾有放量大阳线或涨停板支撑，并在次日根据竞价和盘中信号灵活止盈止损。

能力包含：
- 多技术指标联合筛选（MA、成交量、K线形态）
- 严格的买入条件判断
- 次日卖出规则模拟
- 自动评分排序选出最优股票

触发条件：当用户需要进行量化选股、获取每日股票推荐或执行技术面筛选策略时使用。

## 操作步骤

### 标准流程

**1. 执行选股脚本**

脚本会自动完成以下步骤：
- 获取全市场A股列表
- 下载每只股票的历史日线数据
- 计算技术指标（MA5/MA10/MA20、成交量均线等）
- 筛选满足所有买入条件的股票
- 评分排序输出最强的一只

调用示例：
```bash
# 使用默认参数执行选股（默认当日）
python scripts/stock_quant_select.py

# 指定历史日期执行选股
python scripts/stock_quant_select.py --date 2024-01-15

# 自定义参数
python scripts/stock_quant_select.py --date 2024-01-15 --ma10-diff 0.03 --big-rise-threshold 0.07 --output my_result.json
```

**2. 查看选股结果**

脚本会生成 JSON 格式结果文件，包含：
- 股票代码和名称
- 建议买入价格和时间
- 满足的条件列表
- 技术指标数值
- 综合评分

**3. 次日卖出管理**（由智能体参考规则执行）

根据脚本输出的买入建议，次日执行以下卖出规则：

- **集合竞价止损**：若次日开盘价 < MA10 → 立即卖出
- **盘中止盈**：冲高 2% → 卖出；冲高回落破分时均线 → 卖出
- **强制离场**：10:00 未触发卖出 → 强制卖出
- **特殊情况**：早盘涨停或突破5日线 → 持有至尾盘卖出

### 可选参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| --date | 选股日期（YYYY-MM-DD） | 最新交易日 |
| --output | 输出文件路径 | ./stock_select_result.json |
| --ma10-diff | 10日线回调阈值（±范围） | 0.02（±2%） |
| --big-rise-threshold | 大阳线涨幅阈值 | 0.06（6%） |
| --volume-multiple | 放量倍数 | 1.5 |
| --lookback-days | 历史回看天数 | 20 |
| --limit-up-threshold | 涨停板阈值 | 0.095（9.5%） |

## 策略逻辑说明

### 买入条件（必须全部满足）
1. **当日收阴线**：收盘价 < 开盘价
2. **历史放量大阳线或涨停**：过去20天内存在涨幅≥6%且放量≥1.5倍的交易日
3. **当日缩量**：成交量 < 昨日成交量
4. **回调至10日线**：股价在10日线±2%范围内
5. **MA10向上**：10日线呈上升趋势

### 次日卖出规则
- **开盘**：开盘价 < MA10 → 立即止损
- **盘中**：冲高2%止盈；冲高回落破分时均线止盈；10:00强制卖出
- **特殊**：早盘涨停持有至尾盘；突破5日线持有至尾盘
