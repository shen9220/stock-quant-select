# 如何创建桌面快捷方式

## Windows系统

### 方法1：创建快捷方式

1. 找到 `启动模型.bat` 文件
2. 右键点击该文件
3. 选择"发送到" → "桌面快捷方式"
4. 在桌面上会看到一个快捷方式图标

### 方法2：手动创建快捷方式

1. 在桌面右键，选择"新建" → "快捷方式"
2. 输入位置：`完整路径\stock-quant-select\启动模型.bat`
   - 例如：`C:\Users\你的用户名\Desktop\stock-quant-select\启动模型.bat`
3. 点击"下一步"
4. 输入名称：`股票量化模型`
5. 点击"完成"

## Linux系统

### 方法1：创建桌面快捷方式

1. 打开文件管理器
2. 找到 `启动模型.sh` 文件
3. 右键点击，选择"创建链接"
4. 将链接拖到桌面

### 方法2：创建.desktop文件

1. 在桌面创建新文件 `股票量化模型.desktop`
2. 添加以下内容：

```desktop
[Desktop Entry]
Version=1.0
Type=Application
Name=股票量化模型
Comment=一键运行股票量化模型
Exec=/home/你的用户名/stock-quant-select/启动模型.sh
Icon=terminal
Terminal=true
Categories=Utility;
```

3. 修改文件权限：
```bash
chmod +x ~/桌面/股票量化模型.desktop
```

## Mac系统

### 方法1：创建别名

1. 打开Finder
2. 找到 `启动模型.sh` 文件
3. 右键点击，选择"制作别名"
4. 将别名拖到桌面

### 方法2：创建Automator应用

1. 打开"应用程序" → "Automator"
2. 选择"应用程序" → "选取"
3. 在左侧搜索"运行Shell脚本"，拖到右侧
4. 在脚本框中输入：
```bash
cd /Users/你的用户名/stock-quant-select
./启动模型.sh
```
5. 保存为"股票量化模型.app"

## 每日使用流程

### 第一次使用

1. **创建快捷方式**：按照上述方法创建桌面快捷方式
2. **测试运行**：双击快捷方式，选择模型运行测试
3. **检查依赖**：如果报错，安装必要的依赖库

### 日常使用

1. **双击快捷方式**：点击桌面上的快捷方式图标
2. **选择模型**：输入数字选择要运行的模型
3. **选择日期**：选择今天/昨天/自定义日期
4. **查看结果**：等待程序运行完成，查看分析结果
5. **继续或退出**：选择继续运行其他模型或退出

## 常见问题

### Q1: Windows快捷方式双击没反应？

**解决方案**：
1. 右键点击快捷方式，选择"属性"
2. 检查"目标"路径是否正确
3. 确保 `启动模型.bat` 文件存在
4. 确保Python已安装

### Q2: Linux快捷方式无法运行？

**解决方案**：
1. 确保 `启动模型.sh` 有执行权限：
   ```bash
   chmod +x /path/to/启动模型.sh
   ```
2. 确保 `.desktop` 文件有执行权限：
   ```bash
   chmod +x ~/桌面/股票量化模型.desktop
   ```
3. 检查 `.desktop` 文件中的路径是否正确

### Q3: Mac快捷方式显示"未验证的开发者"？

**解决方案**：
1. 右键点击应用，选择"打开"
2. 在弹出窗口中点击"打开"
3. 或者在系统设置中允许未验证的应用

## 快捷键建议

如果您经常使用，可以设置系统快捷键来快速启动：

### Windows
1. 右键快捷方式 → 属性
2. 在"快捷键"框中输入快捷键
3. 例如：`Ctrl + Alt + S`

### Linux
1. 系统设置 → 键盘 → 快捷键
2. 添加自定义快捷键
3. 命令：`/path/to/启动模型.sh`

### Mac
1. 系统偏好设置 → 键盘 → 快捷键
2. 选择"应用程序快捷键"
3. 添加新快捷键
