## 📋 新增规则

### 1. 目录编号规则

所有主目录使用两位数字前缀：
- `00_` - 必读（最优先）
- `01_` - 核心功能
- `02_` - 工具
- `03_` - 智能体
- `04_` - 文档
- `05_` - 配置
- `06_` - 测试

### 2. 文件关联规则

**正确**:
```
01_量化模型/首板战法/
├── up.py
├── README.md
└── config.json
```

**错误**:
```
01_量化模型/首板战法/up.py
04_文档说明/首板战法文档.md
```

### 3. 新增文件规则

新增文件必须：
1. 放在对应分类目录
2. 与相关文件放在一起
3. 遵循命名规范
4. 包含必要文档
