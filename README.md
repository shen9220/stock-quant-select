## 📊 项目简介

本项目提供多个成熟的量化选股策略，帮助投资者在股票市场中发现机会、控制风险。所有策略都经过严格测试，并提供了完整的交易建议和风险管理方案。

---

## 🎯 核心功能

### 1. 量化模型（01_量化模型/）

- **首板战法**: 首次涨停股票分析，早盘集合竞价买入，次日收盘前卖出
- **涨停三阴法**: 首次涨停后连续3天阴跌不破涨停底部
- **隔夜战法（深度）**: 买阴不买阳，深度选股模型
- **隔夜战法（股票）**: 隔夜选股，快速筛选
- **三阴战法**: 三阴战法选股

### 2. 智能工具（02_智能工具/）

- **一键运行工具**: 图形化界面操作多个模型
- **跨平台支持**: Windows、Linux、Mac

### 3. 智能体（03_智能体/）

- **Stock Assistant**: 智能选股助手，支持自然语言查询
- **技能包**: 集成多个股票分析技能

### 4. 文档系统（04_文档说明/）

- **使用指南**: 快速开始、详细使用说明
- **GitHub文档**: 项目上传和管理
- **技能包文档**: 技能包说明
- **智能体文档**: 智能体使用指南

## 🚀 快速开始

### 环境要求

```bash
Python 3.7+
pip install akshare pandas numpy
```

### 一键运行

```bash
# Linux/Mac
cd /workspace/projects/stock-quant-select/02_智能工具
./启动模型.sh

# Windows
双击 02_智能工具/启动模型.bat
```

### 选择模型

运行后会显示菜单，输入数字选择模型：
```
  [1] 强度首板战法
  [2] 涨停三阴法
  [3] 隔夜战法（深度）
  [4] 隔夜战法（股票）
  [5] 三阴战法
  [6] 运行所有模型
  [0] 退出程序
```
