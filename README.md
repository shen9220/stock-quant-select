# 股票量化选股模型

> 🚀 一套完整的股票量化选股策略系统，包含多个经过验证的交易模型

## 📊 项目简介

本项目提供多个成熟的量化选股策略，帮助投资者在股票市场中发现机会、控制风险。所有策略都经过严格测试，并提供了完整的交易建议和风险管理方案。

## 🎯 核心策略

### 1. 强度首板战法 ⭐
- **文件位置**: `首板战法/up.py`
- **策略逻辑**: 首次涨停股票分析，早盘集合竞价买入，次日收盘前卖出
- **核心优势**: 
  - 综合评分系统（100分制）
  - K线趋势 + 未来走法 + 情绪面 + 基本面分析
  - 提供精确的买卖价格建议

### 2. 涨停三阴法
- **文件位置**: `涨停三阴法/pop3.py`
- **策略逻辑**: 首次涨停后连续3天阴跌不破涨停底部
- **核心优势**:
  - 捕捉涨停后的回调机会
  - 十字星识别技术
  - 强度排名系统

### 3. 隔夜战法（深度）
- **文件位置**: `隔夜战法/deep.py`
- **策略逻辑**: 买阴不买阳，深度选股模型
- **核心优势**:
  - 历史放量大阳线识别
  - 缩量阴线筛选
  - 10日线回调策略

### 4. 隔夜战法（股票）
- **文件位置**: `隔夜战法/stock.py`
- **策略逻辑**: 隔夜选股，快速筛选
- **核心优势**:
  - 实时数据获取
  - 高效筛选机制

### 5. 三阴战法
- **文件位置**: `三阴战法/pop3.py`
- **策略逻辑**: 三阴战法选股
- **核心优势**:
  - 趋势识别
  - 快速入场

## 🚀 快速开始

### 环境要求

```bash
Python 3.7+
pip install akshare pandas numpy
```

### 一键运行

```bash
# 运行一键运行工具
./启动模型.sh

# 或使用 Python
python run_models.py
```

### 单独运行策略

```bash
# 运行强度首板战法
python 首板战法/up.py --date 2024-04-24 --top-n 2

# 运行涨停三阴法
python 涨停三阴法/pop3.py --date 2024-04-24 --top-n 5
```

## 💡 功能特点

- ✅ **一键运行**: 无需命令行，图形化界面操作
- ✅ **多策略支持**: 5个量化策略可选

- ✅ **精确建议**: 提供买入、止盈、止损价格
- ✅ **风险控制**: 风险收益比计算
- ✅ **实时数据**: 使用 akshare 获取最新数据
- ✅ **跨平台**: Windows、Linux、Mac 都支持
