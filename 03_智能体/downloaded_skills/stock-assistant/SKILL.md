# 📈 智能选股助手 (Smart Stock Picker)

> 基于 A 股市场数据的智能选股工具，支持多维度筛选、估值分析、技术指标评分，帮助投资者快速发现优质标的。

## 🎯 能力概述

本 Skill 提供以下核心能力：

| 功能 | 说明 |
|------|------|
| 智能选股 | 根据用户偏好（行业、市值、估值等）筛选符合条件的股票 |
| 估值评分 | 基于 PE/PB 历史百分位评估当前估值水平 |
| 技术评分 | 结合均线、成交量、涨跌幅等指标给出技术面评分 |
| 行业对比 | 将目标股票与同行业公司进行多维度对比 |
| 投资建议 | 综合评分后给出买入/观望/回避建议 |

## 📋 适用场景

- **场景 1**：用户说"帮我选几只低估值的银行股"
- **场景 2**：用户说"我想买新能源，有什么推荐的吗？"
- **场景 3**：用户说"这只股票能买吗？帮我分析一下"
- **场景 4**：用户说"帮我找几只高股息的股票"
- **场景 5**：用户说"最近什么板块比较热门？"

## 🔧 核心功能

### 1. 智能选股引擎

根据用户需求，从以下维度筛选股票：

| 维度 | 说明 | 示例参数 |
|------|------|----------|
| 行业板块 | 按申万行业分类筛选 | 银行、新能源、消费、医药 |
| 市值范围 | 筛选大/中/小盘股 | 大盘(>1000亿)、中盘(200-1000亿)、小盘(<200亿) |
| 估值水平 | PE/PB 历史百分位 | 低估值(<30%)、合理(30-70%)、高估值(>70%) |
| 涨跌幅 | 近期表现筛选 | 近5日涨幅>10%、近20日跌幅>20% |
| 股息率 | 高股息筛选 | 股息率>4%、股息率>6% |
| 技术形态 | K线形态识别 | 金叉、突破、超跌反弹 |

### 2. 估值评分模型

基于历史数据计算估值百分位：

```
估值评分 = 100 - PE百分位 × 0.5 - PB百分位 × 0.3 - PEG偏离度 × 0.2

评分解读：
- 80-100分：显著低估，具备安全边际
- 60-80分：估值合理，可考虑配置
- 40-60分：估值偏高，谨慎参与
- 0-40分：明显高估，建议回避
```

### 3. 技术面评分模型

综合多个技术指标：

```
技术评分 = 均线评分(30%) + 量价评分(25%) + 动量评分(25%) + 形态评分(20%)

均线评分：
- 多头排列 +20分
- 价格站上MA20 +15分
- MA5上穿MA10 +10分

量价评分：
- 放量上涨 +15分
- 缩量回调 +10分
- 量价背离 -10分

动量评分：
- RSI<30(超卖) +15分
- RSI>70(超买) -10分
- MACD金叉 +15分
```

### 4. 综合评级系统

| 评级 | 综合得分 | 建议操作 | 风险等级 |
|------|----------|----------|----------|
| ⭐⭐⭐⭐⭐ | 85-100 | 强烈推荐，重点配置 | 低 |
| ⭐⭐⭐⭐ | 70-84 | 推荐，可适当配置 | 中低 |
| ⭐⭐⭐ | 55-69 | 关注，等待时机 | 中 |
| ⭐⭐ | 40-54 | 谨慎，小仓位试探 | 中高 |
| ⭐ | 0-39 | 回避，风险较大 | 高 |

## 🚀 使用流程

### 标准选股流程

```
用户输入选股需求
    ↓
解析筛选条件（行业/市值/估值等）
    ↓
调用数据接口获取股票列表
    ↓
应用筛选条件过滤
    ↓
计算估值评分 + 技术评分
    ↓
综合评级并排序
    ↓
输出推荐列表 + 投资建议
```

### 个股分析流程

```
用户提供股票代码/名称
    ↓
获取实时行情 + 历史数据
    ↓
计算估值百分位
    ↓
分析技术指标
    ↓
与同行业对比
    ↓
输出分析报告 + 操作建议
```

## ⚠️ 注意事项

### 数据准确性
- 实时数据可能存在 3-5 分钟延迟
- 估值百分位基于历史数据，仅供参考
- 建议结合官方披露的财务数据交叉验证

### 投资风险
- 本 Skill 仅供参考，不构成投资建议
- 股市有风险，投资需谨慎
- 过往业绩不代表未来收益
- 建议分散投资，控制单一标的仓位

### 使用限制
- 单次最多筛选 50 只股票
- 每日最多调用 100 次 API
- 建议在交易时间内使用（9:30-15:00）
