# 操作清单

## ✅ 使用 Stock Assistant 检查清单

### 方法一：在 Coze 平台使用（推荐）

- [ ] **准备工作**
  - [ ] 已注册 Coze 账号
  - [ ] 已登录 https://www.coze.cn/

- [ ] **创建智能体**
  - [ ] 点击"智能体"菜单
  - [ ] 点击"创建智能体"按钮
  - [ ] 输入名称：`我的选股助手`
  - [ ] 输入描述：`智能选股分析助手`
  - [ ] 点击"创建"

- [ ] **添加技能**
  - [ ] 在编辑页面找到"插件/技能"
  - [ ] 点击"添加技能"
  - [ ] 搜索框输入：`stock-assistant`
  - [ ] 找到"Stock Assistant 智能选股助手"
  - [ ] 点击"添加"或"启用"

- [ ] **测试技能**
  - [ ] 在聊天框输入：`帮我选几只银行股`
  - [ ] 等待返回结果
  - [ ] 检查返回结果是否合理

---

### 方法二：本地集成使用

- [ ] **环境准备**
  - [ ] Python 版本 >= 3.7
  - [ ] 已安装 pip

- [ ] **安装依赖**
  - [ ] 运行：`pip install openai-coze`
  - [ ] 运行：`pip install python-dotenv`

- [ ] **复制文件**
  - [ ] 复制 `SKILL.md` 到工作目录
  - [ ] 复制 `README.md` 到工作目录
  - [ ] 复制 `package.json` 到工作目录

- [ ] **配置环境**
  - [ ] 创建 `.env` 文件
  - [ ] 填写 API Key
  - [ ] 填写 Skill ID

- [ ] **创建测试脚本**
  - [ ] 创建 `test_stock_assistant.py`
  - [ ] 编写测试代码
  - [ ] 运行测试
