# Stock Assistant 使用说明

## ⚠️ 重要提示

**您刚才的操作是错误的！**

您在 Linux 命令行中直接输入了：
```
帮我找几只技术面金叉的股票
```

这是错误的！命令行只能执行 Linux 命令，不能直接对话。

---

## ✅ 正确的使用方式

### 方法一：在 Coze 平台使用（最简单）⭐⭐⭐

#### 步骤：

1. **打开浏览器，访问 Coze**
   ```
   https://www.coze.cn/
   ```

2. **登录账号**
   - 使用手机号或邮箱登录

3. **创建智能体**
   - 点击左侧"智能体"
   - 点击"创建智能体"
   - 名称填：`我的选股助手`
   - 点击"创建"

4. **添加 Stock Assistant 技能**
   - 在编辑页面点击"插件/技能"
   - 点击"添加技能"
   - 搜索：`stock-assistant`
   - 点击"添加"

5. **在聊天框输入问题**
   ```
   帮我找几只技术面金叉的股票
   ```

6. **等待返回结果**

---

### 方法二：使用一键运行工具（本地量化模型）

如果您想使用本地的量化模型，可以这样做：

#### 1. 运行一键运行工具

```bash
cd /workspace/projects/stock-quant-select
./启动模型.sh
```

#### 2. 选择模型
```
请选择要运行的模型：

  [1] 强度首板战法          ⭐ 推荐
  [2] 涨停三阴法
  [3] 隔夜战法（深度）
  [4] 隔夜战法（股票）
  [5] 三阴战法
  [6] 运行所有模型
  [0] 退出程序
```

#### 3. 输入数字 1
```
请输入选项（0-6）: 1
```

#### 4. 选择日期
```
请选择日期：
  [1] 今天
  [2] 昨天
  [3] 自定义日期（格式：YYYY-MM-DD）

请输入选项（1-3）:
```

#### 5. 查看结果

程序会自动分析并输出推荐股票。

---

### 方法三：直接运行 Python 脚本

```bash

# 运行涨停三阴法

# 运行隔夜战法

```
