# 快速开始指南

## ⚡ 最快的上手方式（3步）

### 步骤 1：登录 Coze

1. 浏览器打开：https://www.coze.cn/
2. 登录你的账号

### 步骤 2：创建智能体

1. 点击"智能体" → "创建智能体"
2. 名称填：`我的选股助手`
3. 点击"创建"

### 步骤 3：添加技能

1. 在编辑页面点击"插件/技能"
2. 点击"添加技能"
3. 搜索：`stock-assistant`
4. 点击"添加"

### 完成！

现在在聊天框输入：
```
帮我选几只银行股
```
