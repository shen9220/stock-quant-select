# Stock Assistant 操作指南

## 🚀 完整操作流程（从零到使用）

### 方法一：在 Coze 平台使用（推荐）⭐

#### 步骤 1：登录 Coze 平台

1. 打开浏览器，访问：https://www.coze.cn/
2. 点击右上角"登录"
3. 使用手机号或邮箱登录

#### 步骤 2：创建智能体（Bot）

1. 登录后，点击左侧菜单的"智能体"
2. 点击"创建智能体"按钮
3. 填写信息：
   - 智能体名称：`我的选股助手`
   - 功能描述：`智能选股分析助手`
4. 点击"创建"

#### 步骤 3：添加 Stock Assistant 技能

1. 在智能体编辑页面，点击左侧"插件/技能"
2. 点击"添加技能"或"插件"
3. 在搜索框输入：`stock-assistant`
4. 找到"Stock Assistant 智能选股助手"
5. 点击"添加"或"启用"

#### 步骤 4：测试技能

1. 在右侧聊天框输入：
   ```
   帮我选几只银行股
   ```
2. 等待返回结果

## 🔧 故障排除

### 问题 1：找不到技能

**解决方案**：
1. 检查技能名称是否正确：`stock-assistant`
2. 确保技能已发布
3. 联系技术支持

### 问题 2：调用失败

**解决方案**：
1. 检查 API Key 是否有效
2. 检查网络连接
3. 查看错误日志

### 问题 3：返回结果不准确

**解决方案**：
1. 调整问题描述，更具体一些
2. 提供更多上下文信息
3. 多尝试几个问题
