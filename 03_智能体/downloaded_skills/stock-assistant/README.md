# 智能选股助手

基于 A 股市场数据的智能选股工具，支持多维度筛选、估值分析、技术指标评分。

## 触发词

- 选股
- 帮我选股
- 股票推荐
- 分析股票
- 这只股票怎么样

## 使用方法

1. **行业选股**：说"帮我选几只低估值的银行股"
2. **个股分析**：说"帮我分析一下比亚迪"
3. **高股息选股**：说"帮我找几只股息率高的股票"
4. **热门板块**：说"最近什么板块比较热门？"

## 数据来源

- 腾讯财经 API（实时行情）
- 东方财富 API（历史数据）
- 新浪财经 API（财务数据）
