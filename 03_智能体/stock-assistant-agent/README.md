## 🎯 功能介绍

### 1. 智能选股

根据您的需求智能筛选符合条件的股票。

**支持维度**：
- 行业板块：银行、新能源、消费、医药等
- 估值水平：低估值、合理、高估值
- 市值范围：大盘股、中盘股、小盘股
- 技术形态：金叉、突破、超跌反弹

**示例**：
```
帮我选几只低估值的银行股
推荐几只新能源股票
找一些小盘的成长股
```

### 2. 个股分析

对单只股票进行全面分析，给出投资建议。

**分析内容**：
- 估值水平（PE/PB百分位）
- 技术形态（均线、MACD、KDJ）
- 综合评分（0-100分）
- 投资建议（买入/观望/回避）
- 操作建议（买入价、目标价、止损价）

**示例**：
```
分析一下比亚迪能不能买
工商银行怎么样？
帮我分析 600036
```

### 3. 高股息选股

寻找高股息率的优质股票，适合稳健型投资者。

**筛选标准**：
- 股息率 > 4%
- 估值合理偏低
- 财务状况健康

**示例**：
```
找几只股息率高的股票
推荐一些高分红股票
```

### 4. 热门板块分析

分析当前市场热门板块和题材。

**分析内容**：
- 板块涨幅
- 市场热度
- 代表股票

**示例**：
```
最近什么板块比较热门？
什么板块在涨？
```

## ❓ 常见问题

### Q1: 如何重新启动智能体？

**A**: 再次运行启动脚本即可：
```bash
./start.sh
```

### Q2: 智能体支持哪些行业？

**A**: 目前支持以下行业：
- 银行
- 新能源（光伏、风电、锂电）
- 消费（白酒、食品）
- 医药（生物、医疗）

### Q3: 评分系统如何工作？

**A**: 综合评分由以下部分组成：

**估值评分**（100分制）：
```
估值评分 = 100 - PE百分位 × 0.5 - PB百分位 × 0.3 - PEG偏离度 × 0.2
```

**技术评分**（100分制）：
```
技术评分 = 均线评分(30%) + 量价评分(25%) + 动量评分(25%) + 形态评分(20%)
```

**评级标准**：
- ⭐⭐⭐⭐⭐ (85-100分)：强烈推荐
- ⭐⭐⭐⭐ (70-84分)：推荐
- ⭐⭐⭐ (55-69分)：关注
- ⭐⭐ (40-54分)：谨慎
- ⭐ (0-39分)：回避

### Q4: 数据来源是什么？

**A**: 数据来源包括：
- 腾讯财经 API（实时行情）
- 东方财富 API（历史数据）
- 新浪财经 API（财务数据）

### Q5: 如何运行测试？

**A**: 运行测试脚本：
```bash
./run_test.sh
```

或直接运行：
```bash
python3 test.py
```

`config.json` 包含以下配置：

```json
{
  "name": "智能选股助手",
  "version": "1.0.0",
  "config": {
    "trigger_words": ["选股", "帮我选股", "股票推荐", ...],
    "scoring": {
      "valuation": {...},
      "technical": {...}
    },
    "limits": {
      "max_stocks_per_query": 50,
      "max_api_calls_per_day": 100
    }
  },
  "capabilities": {
    "smart_selection": {...},
    "valuation_scoring": {...},
    "technical_scoring": {...},
    "industry_comparison": {...},
    "investment_advice": {...}
  }
}
```
