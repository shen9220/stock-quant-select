# 智能选股助手 - 快速开始（5分钟上手）

## ⚡ 最简单的使用方式

### 第1步：启动智能体

#### Linux/Mac 用户：

```bash
cd /workspace/projects/stock-quant-select/stock-assistant-agent
./start.sh
```

#### Windows 用户：

```bash
cd stock-assistant-agent
start.bat
```

### 第2步：开始对话

启动后，直接输入您的问题：

```
📝 您的问题: 帮我选几只低估值的银行股
```

### 第3步：查看结果

智能体会自动分析并输出推荐股票！
