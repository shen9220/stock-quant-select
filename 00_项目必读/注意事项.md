## 🚫 禁止事项

### 1. 不要随意移动文件

**错误**:
```bash
# 随意移动文件
mv 01_量化模型/首板战法/up.py .
```

**正确**:
```bash
# 遵循目录规范
# 首板战法的所有文件必须在 01_量化模型/首板战法/
```

### 2. 不要在根目录创建文件

**错误**:
```bash
# 在根目录创建文件
touch my_file.txt
```

**正确**:
```bash
# 放在对应分类目录
touch 04_文档说明/使用指南/my_file.txt
```

### 3. 不要泄露敏感信息

**错误**:
```bash
# 提交敏感文件
git add 05_配置文件/.env
git commit -m "add config"
git push
```

**正确**:
```bash
# .env 已在 .gitignore 中
# 只提交模板
git add 05_配置文件/.env.example
git commit -m "add env template"
git push
```

### 4. 不要破坏目录结构

**错误**:
```bash
# 删除分类目录
rm -rf 01_量化模型/
```

**正确**:
```bash
# 只删除具体模块
rm -rf 01_量化模型/废弃模块/
```

### 5. 不要忽略文档

**错误**:
```python
# 添加新功能但不更新文档
def new_feature():
    pass
```

**正确**:
```python
# 添加功能并更新文档
def new_feature():
    """
    新功能说明
    """
    pass

# 同时更新 README.md
```

## 🐛 常见错误及解决

### 错误1：找不到文件

**错误信息**:
```
FileNotFoundError: [Errno 2] No such file or directory: 'up.py'
```

**原因**: 文件路径错误

**解决**:
```bash
# 检查文件位置
find . -name "up.py"

# 使用正确路径
cd 01_量化模型/首板战法/
python up.py
```

### 错误2：权限不足

**错误信息**:
```
Permission denied: 'script.sh'
```

**原因**: 没有执行权限

**解决**:
```bash
chmod +x script.sh
./script.sh
```

### 错误3：编码错误

**错误信息**:
```
UnicodeDecodeError: 'utf-8' codec can't decode
```

**原因**: 文件编码问题

**解决**:
```python
# 使用正确的编码
with open(file, 'r', encoding='utf-8') as f:
    content = f.read()
```

### 错误4：导入错误

**错误信息**:
```
ModuleNotFoundError: No module named 'xxx'
```

**原因**: 缺少依赖包

**解决**:
```bash
pip install xxx
```
